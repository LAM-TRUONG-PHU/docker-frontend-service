Yêu cầu:
  Cần tải docker desktop và docker compose
Các bước start dự án:
  Bước 1: git clone (https://github.com/LAM-TRUONG-PHU/docker-frontend-service.git)
  Bước 2: cd frontend
  Bước 3: chạy lệnh `npm install`
  Bước 4: cd ..
  Bước 5: chạy lệnh docker compose up --build 
  Bước 6: Nhập vào link: http://localhost/
